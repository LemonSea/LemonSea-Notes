# Node Server 的特点

Node Server 偏底层，如果不适用框架，大多的功能需要自己写代码实现。

Node  开启的 Web 服务，是一个完全的黑盒子，不像 php 中，无论代码还是网页，都可以通过 web url 路径来访问。在 Node 中，所有资源默认都不允许用户访问，用户可以访问的资源由开发人员编写代码指定。                                                                                                 

Node 的自定义非常灵活。

# 1. Express 是什么

- Express 是一个第三方的 Web 开发框架
- 高度封装了 http 模块
- 更加专注于业务，而非底层细节（Node.js 就是很底层。^ - ^ 我喜欢底层~~）

# 2.  Express 初试

```js
var express = require('express')

// 创建服务
// 同 http.createServer()
let app = express()

// 当服务器收到 get 请求 / 的时候，会执行回调函数
app.get('/', (req, res) => {
    res.send('Hello Express!')
})

// 同 server.listen
app.listen(3000, () => {
    console.dir('server is running at prot 3000')
})
```
